# TaskFlow

A project management application with real-time collaboration features. Built with React, Node.js, MongoDB, Socket.io, and Redux.

## Features

- Real-time Collaboration with Socket.io
- Kanban Board with drag-and-drop
- Team Workspaces
- Task Assignment with deadlines
- Role-based Access Control

## Tech Stack

- Frontend: React 18, Redux Toolkit, React DnD, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB with Mongoose
- Real-time: Socket.io
- Auth: JWT + bcrypt

## Getting Started

```bash
git clone https://github.com/hannan-rizwan/TaskFlow.git
cd TaskFlow
npm run install-all
cp .env.example .env
npm run dev
```
